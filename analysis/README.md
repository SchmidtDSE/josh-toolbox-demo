# Analysis Outputs

This directory contains **derived analysis artifacts** from simulation results.

## Directory Structure

```
analysis/
├── figures/
│   ├── inputs/          # Input data visualizations (Python)
│   ├── timeseries/      # Time series plots (Python + R)
│   ├── spatial/         # Spatial analysis maps (Python + R)
│   └── comparisons/     # Scenario comparison figures (R)
├── summaries/           # Aggregated data tables
├── visualizations.R     # R script for publication figures
└── spatial_viz.py       # Legacy Python visualization script
```

## Regenerating Files

Run the full `demo_workflow.ipynb` notebook:
1. **Steps 1-7**: Generate Python figures and export data
2. **Step 8**: Call `Rscript analysis/visualizations.R` for publication figures

The R script is called automatically from the notebook - no separate execution needed.
